HOW TO UPLOAD:
-------------

Method 1: FTP/SFTP (Recommended)
1. Open FileZilla or your FTP client
2. Connect to gilafstore.com
3. Navigate to: /public_html/event-manager/
4. Upload all files maintaining the folder structure

Method 2: cPanel File Manager
1. Login to cPanel
2. Open File Manager
3. Navigate to: public_html/event-manager/
4. Upload files one by one or as a zip

Method 3: Git (If using version control)
1. Commit all changes locally
2. Push to your repository
3. Pull on production server
