## ✅ HOW IT WORKS

The Event Manager automatically detects whether you're on:
- **Localhost:** Uses `http://localhost/gilafstore.com/public_html/`
- **Production:** Uses `https://gilafstore.com/`

**No manual URL changes needed!** Just use the same files on both environments.

---

## 🚀 QUICK START

### On Localhost (XAMPP)
1. Start Apache and MySQL in XAMPP
2. Open: `http://localhost/gilafstore.com/public_html/event-manager/migrations/install.php`
3. Click "Run Installation"
4. Done!

### On Production (Hostinger)
1. Upload files to server
2. Open: `https://gilafstore.com/event-manager/migrations/install.php`
3. Click "Run Installation"
4. Done!
